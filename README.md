# Учебный fullstack-проект для практики от компании Zuzex.

Веб-сервис – социальная сеть ИТ-специалистов

## Основные модули
- Авторизация
- Главная страница с постами
- Профиль пользователя
- Портфолио проектов

## Стек
Frontend: React, Vite, TypeScript, Redux, SCSS  
Backend: Node.js, NestJS, MongoDB, Mongoose  
Auth: JWT, bcrypt, HttpOnly cookies

## Соответствие ТЗ

### Закрыто
- Авторизация: регистрация, вход, роли, cookies-сессия, серверная валидация
- Основная страница: лента постов, фильтрация, создание, редактирование, удаление
- Лайки постов
- Markdown-редактор постов и preview
- Фото-превью постов
- Профиль пользователя: просмотр своего и чужого профиля
- Редактирование профиля
- Портфолио: добавление, редактирование, удаление проектов
- Модальное окно проекта
- Фото-превью проектов
- Компонентная декомпозиция профиля по ТЗ
- Эндпоинты профиля и портфолио в формате `/users/:id/...`
- Gitflow с ветками `main`, `dev`, `feature/*`

### Не закрыто полностью
- Специальные бизнес-возможности ролей `HR` и `Manager`
- Merge Request / Pull Request через GitHub UI не использовались, слияния выполнялись локально через git
